Sume, Produse, Numărări

Probleme axate pe calculul sumelor (Gauss, pătrate), produselor (factoriale) și numărării elementelor cu anumite proprietăți.

Grilă Formula lui Gauss

Care este formula corectă pentru calcularea sumei numerelor naturale de la 1 la n ($S = 1 + 2 + ... + n$)?

Algoritm #46 SumaPare Ușoară

Se dă numărul natural n. Calculați suma primelor n numere naturale pare nenule.

Exemplu: Pentru n = 3, suma este 2 + 4 + 6 = 12.

Indicație: Matematic, suma este $2 \times (1 + 2 + ... + n) = n \times (n+1)$.


#include <iostream>
using namespace std;

int main() {
    long long n;
    cin >> n;
    // Putem folosi formula directă pentru eficienta
    // Suma = 2 + 4 + ... + 2n = 2*(1+2+..+n) = n*(n+1)
    cout << "Suma este " << n * (n + 1);
    return 0;
}
Algoritm #49 Factorial Ușoară

Să se calculeze produsul primelor n numere naturale (n!).

Exemplu: n = 4 => 1 * 2 * 3 * 4 = 24.


#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    // Folosim long long deoarece factorialul creste foarte repede
    unsigned long long P = 1; 
    
    if (n == 0) {
        cout << 1;
    } else {
        for(int i = 1; i <= n; i++) {
            P = P * i;
        }
        cout << P;
    }
    return 0;
}

Divizibilitate

Probleme despre divizori, numere prime, CMMDC/CMMMC și descompunere în factori.

Grilă Numere Prime

Care dintre următoarele afirmații despre un număr prim p este adevărată?

Algoritm CMMDC (Algoritmul lui Euclid) Ușoară/Medie

Se dau două numere naturale a și b. Să se determine Cel Mai Mare Divizor Comun al lor.

Exemplu: a = 12, b = 18 -> CMMDC = 6.


#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    
    // Algoritmul lui Euclid prin impartiri repetate
    while(b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    // La final, CMMDC se afla in 'a'
    cout << a;
    return 0;
}
Algoritm #45 VerifPrim Ușoară

Verifică dacă un număr n este prim.


bool esteprim(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    
    // Verificam divizorii impari pana la radical din n
    for (int d = 3; d * d <= n; d += 2) {
        if (n % d == 0) return false;
    }
    return true;
}

Cifrele unui număr

Prelucrarea cifrelor: extragere, maxime, oglindit, palindromuri, eliminare cifre.

Grilă Ultima cifră

Care expresie returnează ultima cifră a unui număr întreg x?

Algoritm #126 CifraMaxima Ușoară

Se citește un număr natural n. Să se afișeze cifra maximă a acestuia.

Exemplu: n = 274 -> Cifra maximă este 7.


#include <iostream>
using namespace std;

int main() {
    int n, maxCif = -1;
    cin >> n;
    
    if (n == 0) maxCif = 0;
    
    while(n != 0) {
        int cifra = n % 10; // Ultima cifra
        if (cifra > maxCif) {
            maxCif = cifra;
        }
        n = n / 10; // Taiem ultima cifra
    }
    cout << maxCif;
    return 0;
}
Algoritm #1391 Oglindit Ușoară

Calculați oglinditul unui număr (numărul scris invers). Dacă numărul este egal cu oglinditul său, se numește palindrom.

Exemplu: 123 -> 321.


#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int ogl = 0, copie = n;
    
    while(n != 0) {
        ogl = ogl * 10 + n % 10;
        n = n / 10;
    }
    
    cout << "Oglinditul este: " << ogl << endl;
    if (ogl == copie) cout << "Este palindrom!";
    else cout << "Nu este palindrom.";
    
    return 0;
}
