Algoritm Poziție și Valoare în Pătrat

Se consideră un pătrat nxn numerotat de la 1 la n^2. Determinați:
1. Valoarea V la poziția (x,y).
2. Poziția (i,j) a valorii val.

Exemplu: Intrare: 4 2 3 12 -> Ieșire: 7 3 4 

#include <iostream>
using namespace std;
int main() {
    long long n, x, y, val;
    cin >> n >> x >> y >> val;
    // 1. Valoarea la x, y
    // (x-1) linii complete + y coloane
    cout << (x - 1) * n + y << " ";
    
    // 2. Pozitia valorii val
    long long i, j;
    if (val % n == 0) {
        i = val / n;
        j = n;
    } else {
        i = val / n + 1;
        j = val % n;
    }
    cout << i << " " << j;
    return 0;
}

    int n, v[3];

cout << endl;
// Linia 2: div 3

cout << endl;
// Linia 3: (div 2 sau div 3) si ! div 6
